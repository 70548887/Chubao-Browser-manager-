# Pencil 原型设计指南 - 触宝指纹浏览器

> 使用 Pencil 进行可视化界面设计，快速迭代 UI 原型

---

## 一、Pencil 简介

**Pencil** 是一款开源的原型设计工具，支持：
- ✅ 拖拽式界面设计
- ✅ 丰富的 UI 组件库
- ✅ 导出为 PNG/PDF/HTML
- ✅ 跨平台（Windows/Mac/Linux）

---

## 二、快速开始

### 2.1 启动 Pencil

1. 打开 Pencil 应用程序
2. 选择 **File → New Document**
3. 选择模板：**Desktop Application** 或 **Web Page**

### 2.2 设置画布尺寸

我们的应用是桌面应用，推荐尺寸：

```
宽度：1280px
高度：800px
背景色：#1e1e1e (深色主题)
```

**操作步骤**：
1. 右键点击画布
2. 选择 **Page Properties**
3. 设置 **Width** 和 **Height**
4. 设置 **Background Color**

## 三、设计我们的界面

### 3.1 主布局结构

我们的应用采用经典的三栏布局：

```
┌─────────────────────────────────────────────────┐
│  自定义标题栏 (32px)                              │
├──────────┬──────────────────────────────────────┤
│          │  工具栏 (60px)                        │
│  侧边栏   ├──────────────────────────────────────┤
│  (240px) │                                       │
│          │  主内容区                              │
│          │  (环境列表)                            │
│          │                                       │
└──────────┴──────────────────────────────────────┘
```

### 3.2 使用 Pencil 组件

#### **步骤 1：添加容器**

1. 从左侧工具栏拖拽 **Rectangle** 到画布
2. 设置属性：
   - **Width**: 1280px
   - **Height**: 800px
   - **Fill Color**: #1e1e1e
   - **Border**: None

#### **步骤 2：创建标题栏**

1. 拖拽 **Rectangle**
2. 设置属性：
   - **Width**: 1280px
   - **Height**: 32px
   - **Fill Color**: 渐变 (#409eff → #1e5bb8)
   - **Position**: X=0, Y=0

3. 添加文本 **Text**：
   - 内容：`触宝指纹浏览器 v0.1.0`
   - 颜色：白色 (#ffffff)
   - 字体：14px, Bold

4. 添加窗口控制按钮（使用 **Button** 组件）：
   - 最小化：`—`
   - 最大化：`□`
   - 关闭：`✕`

#### **步骤 3：创建侧边栏**

1. 拖拽 **Rectangle**
2. 设置属性：
   - **Width**: 240px
   - **Height**: 768px (800 - 32)
   - **Fill Color**: #2c2e33
   - **Position**: X=0, Y=32

3. 添加菜单项（使用 **List** 或 **Button**）：
   ```
   📊 运行中 [20]
   📁 分组管理
   🔗 代理管理
   🎛️ 窗口群控
   ⏱️ RPA 自动化
   ⚙️ 系统设置
   ```

#### **步骤 4：创建主内容区**

1. 拖拽 **Table** 组件
2. 设置列：
   - ☑️ 选择框
   - 🏷️ 序号
   - 📝 分组
   - 🔖 标签
   - 🌐 国口域名
   - 📱 代理组组
   - ⏰ 时间
   - 🔧 操作

## 四、Element Plus 组件映射

Pencil 中的组件对应我们实际使用的 Element Plus 组件：

| Pencil 组件 | Element Plus 组件 | 用途 |
|------------|------------------|------|
| **Rectangle** | `<div>` | 容器、背景 |
| **Button** | `<el-button>` | 按钮 |
| **Text Input** | `<el-input>` | 输入框 |
| **Checkbox** | `<el-checkbox>` | 复选框 |
| **Table** | `<el-table>` | 数据表格 |
| **List** | `<el-menu>` | 菜单列表 |
| **Badge** | `<el-badge>` | 徽章 |
| **Icon** | `<el-icon>` | 图标 |

## 五、设计技巧

### 5.1 使用网格对齐

1. 启用网格：**View → Show Grid**
2. 设置网格间距：**8px**（符合 8px 设计规范）
3. 启用吸附：**View → Snap to Grid**

### 5.2 使用颜色变量

创建颜色样本，保持一致性：

```
主色调：#409eff (蓝色)
成功色：#67c23a (绿色)
警告色：#e6a23c (橙色)
危险色：#f56c6c (红色)

背景色：
  - 主背景：#1e1e1e
  - 卡片背景：#2c2e33
  - 边框：#363b45

文字色：
  - 主文字：#e5eaf3
  - 次文字：#a3a6ad
  - 禁用：#606266
```

### 5.3 组件复用

1. 设计好一个菜单项后，右键选择 **Duplicate**
2. 修改文本内容
3. 保持间距一致（8px 或 16px）

## 六、导出和交付

### 6.1 导出为图片

1. 选择要导出的页面
2. **File → Export → PNG**
3. 保存到 `docs/designs/` 目录

### 6.2 标注尺寸

在 Pencil 中添加标注：
1. 使用 **Line** 工具绘制标注线
2. 使用 **Text** 添加尺寸说明
3. 例如：`240px` `32px` `16px padding`

### 6.3 交付给开发

导出后的设计稿应包含：
- ✅ 完整的界面截图
- ✅ 尺寸标注
- ✅ 颜色代码
- ✅ 字体大小
- ✅ 间距说明

## 七、实战案例：设计环境列表页

### 步骤 1：创建新页面

1. **File → New Page**
2. 命名：`环境列表页`

### 步骤 2：添加工具栏

```
┌─────────────────────────────────────────────────┐
│ 共 50 个环境 | 运行中 20    [筛选] [批量操作] [搜索] │
└─────────────────────────────────────────────────┘
```

1. 拖拽 **Rectangle** (背景)

### 步骤 3：添加表格

1. 拖拽 **Table** 组件
2. 设置列数：8 列
3. 设置行数：10 行（示例数据）

### 步骤 4：添加操作按钮

在表格最后一列添加：
- 🚀 启动按钮（蓝色）
- ⏸️ 停止按钮（灰色）
- ✏️ 编辑按钮
- 🗑️ 删除按钮

## 八、从 Pencil 到代码

设计完成后，按照以下流程转换为代码：

```vue

  <div class="stats">共 50 个环境</div>

  <el-button-group>...</el-button-group>
```

```scss
// 从 Pencil 设计稿提取

  padding: 16px;
```

```vue
  style="background-color: #409eff"

  批量启动

```

## 九、常见问题

### Q1: Pencil 中没有我需要的组件怎么办？

**A**: 使用基础组件组合：
- 用 **Rectangle** + **Text** 组合成自定义按钮
- 用 **Rectangle** + **Icon** 组合成图标按钮
- 用多个 **Rectangle** 组合成复杂布局

### Q2: 如何保持设计一致性？

**A**: 
1. 创建组件模板（Templates）
2. 使用颜色样本（Color Swatches）
3. 定义间距规范（8px/16px/24px）

### Q3: 设计稿和实际效果不一致？

**A**: 
1. 使用浏览器开发者工具测量实际尺寸
2. 调整 Pencil 中的尺寸
3. 保持设计稿和代码同步更新

## 十、推荐工作流

```
1. 在 Pencil 中设计原型
   ↓
2. 导出设计稿 (PNG)
   ↓
3. 标注尺寸和颜色
   ↓
4. 根据设计稿编写 Vue 组件
   ↓
5. 在浏览器中对比效果
   ↓
6. 调整代码或设计稿
   ↓
7. 重复 5-6 直到满意
```
