# 项目文件结构 - 触宝指纹浏览器

> 根目录: `browser-manager/`
> 遵循 Domain-Driven Design (DDD) 或 Feature-based 结构。

## 1. 根目录结构
```text
browser-manager/
├── src-tauri/                  # [Rust] 后端应用
├── src/                        # [Vue] 前端应用
├── logs/                       # [Runtime] 运行日志目录 (自动生成)
├── scripts/                    # [DevOps]CI/CD 脚本、数据库迁移脚本
├── .env.development            # [Config] 开发环境变量
├── .env.production             # [Config] 生产环境变量
├── Cargo.toml                  # Rust Workspace 配置
└── package.json                # 前端依赖配置
```

## 2. 后端结构 (src-tauri)
```text
src-tauri/
├── src/
│   ├── config/                 # [Config] 静态配置加载
│   │   ├── mod.rs
│   │   └── app_config.rs       # 映射 .env 或 config.toml
│   ├── modules/                # [Features] 功能模块
│   │   ├── profile/            # 环境管理模块
│   │   │   ├── mod.rs
│   │   │   ├── service.rs      # 业务逻辑
│   │   │   └── entity.rs       # 数据库模型
│   │   ├── browser/            # 浏览器控制模块
│   │   │   ├── mod.rs
│   │   │   ├── launcher.rs     # 启动器
│   │   │   └── supervisor.rs   # 进程监控
│   │   └── proxy/              # 代理模块
│   │       └── bridge.rs       # TCP 转发器
│   ├── infrastructure/         # [Shared] 基础设施
│   │   ├── database/           # SQLite 连接池
│   │   ├── logger/             # Tracing 配置
│   │   └── crypto/             # AES 加密工具
│   ├── api/                    # [Interface] Tauri Commands
│   │   └── mod.rs              # 统一导出 commands
│   ├── main.rs                 # 应用程序入口
│   └── lib.rs                  # Lib 入口
├── tests/                      # [Tests] 集成测试
│   ├── proxy_test.rs
│   └── profile_test.rs
└── migrations/                 # [DB] SQL 迁移文件
```

## 3. 前端结构 (src)
```text
src/
├── api/                        # [Network] API 封装
│   ├── profile.ts
│   └── system.ts
├── assets/                     # [Static] 静态资源
│   └── styles/                 # 全局样式 (Tailwind/SCSS)
├── components/                 # [Shared] 公共组件
│   ├── common/                 # 原子组件 (Button, Input)
│   └── layout/                 # 布局组件 (Sidebar, Header)
├── composables/                # [Logic] 组合式函数
│   └── useTheme.ts
├── config/                     # [Config] 前端常量配置
│   └── index.ts                (NO hardcoding urls here)
├── features/                   # [Features] 业务功能模块
│   ├── dashboard/              # 仪表盘模块
│   │   ├── components/         # 模块私有组件
│   │   ├── store/              # 模块状态
│   │   └── DashboardView.vue   # 页面入口
│   ├── fingerprint/            # 指纹配置模块
│   └── settings/               # 设置模块
├── router/                     # [Router] 路由配置
├── types/                      # [Types] TypeScript 类型定义
├── App.vue
└── main.ts
```

## 4. 关键文件说明
*   `src-tauri/src/config/app_config.rs`: 负责读取所有环境变量，禁止在代码其他地方直接读取 `std::env`。
*   `src/config/index.ts`: 前端所有常量的聚集地，如默认超时时间、重试次数。
*   `src-tauri/migrations/`: 数据库 Schema 变更记录，保证团队开发一致性。
